# xmd_wave
> 波纹动效

### 请xmd_wave元件放置在页面的[0,0]位置；任意元件触发fireMe按钮鼠标点击事件即可产生波纹动效；

### 如果xmd_wave被动态面板嵌套,那么请设置【勿删！请参照说明移动】元件的x，y位置为xmd_wave相对于页面原点的x，y位置。例如页面中xmd_wave位于页面绝对位置(100,200),那么请将【勿删！请参照说明移动】元件设置x=100,y=200。
